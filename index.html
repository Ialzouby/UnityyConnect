UnityConnect
Your Bridge to Community and Belonging

UnityConnect is a supportive digital space designed specifically for refugees and immigrants, helping them settle into their new communities through language support, resource connections, and cultural integration. This platform not only breaks down language barriers but also connects users to vital resources and social events, empowering them to feel welcomed and informed.

Get In Touch

Understanding the Need

Every year, refugees and immigrants face significant challenges as they settle into new communities. From language barriers and unfamiliar transit systems to limited access to essential resources, these obstacles can make integration overwhelming. UnityConnect is designed to bridge these gaps, providing tools that make it easier to find support, build connections, and feel at home.

Real-Time Translation for Conversations

This feature enables seamless communication by providing in-app messaging with automatic translation. Users can chat with locals and other immigrants in their preferred language, helping them seek assistance, connect with their community, and build relationships without language barriers. By supporting diverse languages, UnityConnect fosters an inclusive environment for everyone.

Resource Directory with Maps

A comprehensive resource directory shows the locations of vital services, such as healthcare centers, shelters, community centers, and social services offices. Each entry includes operating hours, contact details, and a brief description of available services. This feature helps newcomers quickly locate essential resources in their area, making it easier to adapt to a new environment.

Cultural and Community Events

This feature showcases upcoming events designed for cultural exchange and community building. Users can discover and participate in language practice meetups, cultural fairs, and local support sessions, creating opportunities for integration and connection. By joining these gatherings, newcomers can make friends, learn about local customs, and share their own culture.

Read More

Support Chat with Volunteer Mentors

UnityConnect partners with local volunteers to provide one-on-one support for newcomers. Through in-app messaging, users can connect with mentors who offer guidance on daily life, job opportunities, transportation, healthcare, and more. This feature ensures that users have access to trusted support as they navigate life in a new place.

Emergency and Legal Support Information

This feature includes a dedicated section for urgent and essential resources, offering contacts for emergency services, legal aid for immigration issues, and mental health support. By having quick access to critical information, users are better equipped to handle challenging situations, promoting a sense of security and peace of mind.

Read More

Language Exchange Pairing

UnityConnect connects users interested in learning each other’s languages, creating a mutual exchange that benefits both parties. By practicing language skills together, refugees and locals can build friendships, strengthen language confidence, and gain a deeper understanding of each other’s cultures. This feature supports community connection and language learning in an interactive way.

Read More

Transportation Resources

This feature provides essential information on local transportation options, including public transit schedules, affordable rideshare options, and biking routes. Users can access guides on navigating local transit systems, helping them reach work, school, and community events with ease. Links to transit apps and route planners are included, making it simpler for newcomers to explore their new community confidently and independently.

Read More

AI Chat Assistant

UnityConnect includes a smart, multilingual AI Chat Assistant designed to answer questions in the user's chosen language. This powerful tool can help users navigate the app, offering guidance on finding specific resources and services within UnityConnect. Additionally, it draws on information from the web to answer a wide range of questions, ensuring users get relevant, timely support for both the app and general inquiries. Whether users need help understanding local customs, finding language resources, or exploring community events, the AI Chat Assistant is there to support them every step of the way.

Project Setup

Frontend

  • Xcode: Developed with Swift and SwiftUI for a seamless user experience.
  • SwiftUI: Elegant UI design and animations for smooth navigation.
  • MapKit: Apple's native map API used for location-based features.

Backend

  • Node.js: Backend server handling real-time API requests for ChatGPT integration.
  • Firebase: Comprehensive solution for authentication, database management, and hosting.

Other Features

Emergency & Legal Support

Quick access to emergency and legal support information, ensuring user safety and support during critical situations. This feature helps users find immediate assistance when needed.

Firebase Authentication

Secure and reliable user authentication using Firebase, allowing users to sign up, sign in, and maintain persistent sessions for a safe and convenient experience.

Firebase Database & Hosting

User data, chat history, and event details are securely stored using Firebase Firestore. The app is hosted on Firebase Hosting for robust performance and reliability.
